# 📈 나의 주식 매매 일지 (Stock Trading Journal)

개인 투자자를 위한 **로컬 기반 주식 매매 일지 및 포트폴리오 관리 웹 애플리케이션**입니다.
파이썬(Flask) 백엔드와 가벼운 SQLite 데이터베이스를 사용하여 모든 데이터를 사용자의 PC에 안전하게 보관합니다.

## ✨ 주요 기능

- **📝 매매 일지 & 메모 작성**: 매수, 매도, 관망 기록뿐만 아니라 종목에 대한 일반 메모와 아이디어를 기록할 수 있습니다.
- **📊 포트폴리오 대시보드**: 작성된 매매 기록을 바탕으로 현재 보유 종목, 총 투자 금액, 평균 단가, 누적 실현 손익을 자동 계산하고 파이 차트(Chart.js)로 시각화합니다.
- **📅 캘린더 뷰**: 월별 달력에서 일자별 매매 건수, 메모 건수, 실현 손익을 한눈에 파악할 수 있습니다.
- **📰 실시간 뉴스 연동**: 현재 보유 중이거나 최근 기록한 종목과 관련된 실시간 뉴스를 구글 뉴스(RSS)에서 가져와 우측 사이드바에 5분 주기로 업데이트합니다.
- **🖼️ 이미지 첨부**: 차트나 호가창 스크린샷을 첨부할 수 있으며, 최적화를 위해 별도의 로컬 폴더(`uploads/`)에 안전하게 저장됩니다.
- **💾 CSV 가져오기 / 내보내기**: 작성한 데이터를 엑셀 등에서 볼 수 있도록 CSV 형태로 백업하거나 불러올 수 있습니다.

<br>

## 🚀 설치 및 실행 방법

### 1. 필수 요구 사항
- Python 3.x 버전
- 웹 브라우저 (Chrome, Safari, Edge 등)

### 2. 라이브러리 설치
터미널 또는 명령 프롬프트를 열고 아래 명령어를 입력하여 필요한 패키지를 설치합니다.
```bash
pip install flask
```

### 3. 서버 실행
프로젝트 폴더로 이동한 뒤, 아래 명령어를 통해 로컬 서버를 실행합니다.
```bash
python app.py
```

### 4. 접속
웹 브라우저를 열고 다음 주소로 접속합니다.
```text
http://127.0.0.1:5000
```

<br>

## 📁 프로젝트 구조

```text
stock-memo/
├── app.py             # 백엔드 서버 구동 및 DB 관리 로직 (Flask)
├── stock-memo.html    # 프론트엔드 메인 화면 구조 (HTML)
├── style.css          # 화면 디자인 및 레이아웃 (CSS)
├── script.js          # 화면 동작, 데이터 통신, 차트 로직 (JavaScript)
├── db/                # 데이터베이스 폴더
│   └── journal.db     # 자동 생성되는 매매 기록 데이터베이스 파일 (SQLite)
└── uploads/           # 첨부된 이미지 파일이 저장되는 폴더
```

<br>

## 🛠️ 유지보수 및 데이터 백업
- 모든 매매 데이터는 프로젝트 폴더 내의 `db/journal.db` 파일에 저장됩니다.
- 이미지는 `uploads/` 폴더에 저장됩니다.
- PC를 옮기거나 포맷할 때는 위 두 항목(`db/` 폴더, `uploads/` 폴더)만 백업해 두시면 모든 기록을 그대로 복구할 수 있습니다.
